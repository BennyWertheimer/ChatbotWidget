SealX Chat Widget — Demo

This page tests the widget embed. Use either method below.

Option 1: Script tag (injects iframe)

Add this to your page (replace the API URL with your backend):

<script src="https://YOUR-HOST/widget.js" data-api-url="https://YOUR-API" data-workspace="SealX" data-theme-color="#111"></script>

For local demo, the widget will load below using the script tag.

Option 2: Iframe

<iframe src="https://YOUR-HOST/embed.html?apiBaseUrl=https://YOUR-API" title="SealX Chat" width="400" height="600" style="border:none;"></iframe>
